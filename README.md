# End-to-End Stock Trading Platform

An end-to-end stock trading web application built using the MERN stack (React, Node.js, Express, MongoDB).

## Features
- User authentication (Login/Register)
- Buy and Sell stocks
- Portfolio management
- Transaction history
- Secure REST APIs

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Project Structure

client/ → React frontend
server/ → Node + Express backend


## How to Run
1. Clone the repo  
2. Install dependencies  
3. Start frontend and backend servers
